Added an opt-in `terminal-tinted` theme that keeps the default terminal theme unchanged while allowing terminals to remap palette slots `16..23` for tinted Pi UI colors:

- selected item background
- tool error, success, and pending backgrounds
- user and custom message backgrounds
- tool title and output text
