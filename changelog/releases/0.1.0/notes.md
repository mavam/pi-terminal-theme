## 🚀 Features

### Add terminal theme

Added a `terminal` theme for pi. Why?

- Keeps pi aligned with your terminal's existing look.
- Avoids a mismatched color experience between pi and the rest of your terminal apps.

How to use:

```bash
pi install npm:@mavam/pi-terminal-theme
```

Then select `terminal` in `/settings` under `theme`.
